# HR Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive **HR Dashboard** built in Power BI using the cleaned HR dataset.  
The dashboard provides insights into workforce metrics through three main sections:
- **Executive Overview**
- **Distribution Analysis**
- **Performance Analysis**

---

## 📊 Executive Overview
- **Measures Created**: Total Employees, Avg Performance, Employees with Score A, Avg Tenure, Total Salary Paid, Median Salary  
- **Calculated Columns**: Age Group, Tenure, Performance Score Numeric  
- **Visuals**:
  - 6 KPI Cards:
    - Total Employees = 1000  
    - Avg Performance = 3.02  
    - Employees with Score A = 19.2%  
    - Avg Tenure = 13.6 years  
    - Total Salary Paid = $50.16M  
    - Median Salary = $60K  
  - Gauge: Target Performance Score (Min=0, Current=3.02, Target=4.25, Max=5)  
  - Pie Chart: Employees in each department  
  - Bar Chart: Gender distribution by department  
  - Tile Chart: Employees by position  
  - Line Chart: Hiring trend  
  - Slicers: Department, Gender, Position, Age Group  

---

## 📊 Distribution Analysis
- **Visuals**:
  - Matrix: Department by position  
  - Bar Chart: Salary band by department  
  - Clustered Column Chart: Gender by age group  
  - Scatter Plot: Salary distribution by department  
  - Decomposition Tree: Employees by department  
  - Slicers: Department, Gender, Position, Age Group  

---

## 📊 Performance Analysis
- **Visuals**:
  - 3 KPI Cards:
    - Excellent Performance = 19.20%  
    - Good Performance = 41.70%  
    - Poor Performance = 39.10%  
  - Gauge: Avg Performance (Min=0, Current=3.02, Target=4.25, Max=5)  
  - 100% Stacked Bar Chart: Performance distribution by department  
  - Clustered Column Chart: Performance score breakdown  
  - Scatter Plot: Salary vs Performance  
  - Clustered Bar Charts: Performance by age group, Performance by gender  
  - Matrix: Percentage of employees (Department by performance score)  
  - Table: First 10 entries  

---

## 🎯 Purpose
This dashboard provides HR executives with a comprehensive view of workforce distribution, salary structures, and performance trends.  
It enables data-driven decision making by combining **overview KPIs**, **distribution insights**, and **performance analysis** in one interactive report.
