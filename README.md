# Power BI Projects Repository

## 📌 Overview
This repository is a collection of my **Power BI projects**.  
Each project is organized into its own folder and contains:
- **Dataset** – the raw or cleaned data used for the dashboard  
- **Power BI Dashboard (.pbix)** – the interactive report file  
- **Images** – screenshots of the dashboard for quick preview  
- **Assets/Icons** – any custom visuals, icons, or supporting files  

---

## 🎯 Purpose
The goal of this repository is to:
- Keep all Power BI projects in one place  
- Maintain a clear and consistent folder structure  
- Provide easy navigation for datasets, dashboards, and visuals  
- Showcase data analysis and visualization work  

---

## 🚀 How to Use
- Navigate to the project folder you’re interested in.  
- Open the dataset to explore the raw/cleaned data.  
- Open the `.pbix` file in Power BI Desktop to interact with the dashboard.  
- View the `images/` folder for screenshots of the dashboard.
